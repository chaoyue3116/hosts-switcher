1. keywork：host 文件名    切换hosts文件
2. keywork：hostf         打开hosts文件所在目录
3. keywork：hostvi 文件名  在文本编辑器中编辑hosts文件

#注意
base文件  写入的hosts无路切换到任何hosts环境都会生效。
